# 🪨 Token Saver Prompt — Caveman Language for AI

> Speak less. Save more. Cut AI token usage by up to 70%.

## The Prompt

```
Respond in caveman language. Only necessary words. No filler, no preamble, no pleasantries. Just the answer.
```

## Why It Works

LLMs (like Claude, ChatGPT) are verbose by default — trained to sound helpful and polite. That costs tokens.

**Caveman mode cuts:**
- Preamble ("Great question! I'd be happy to help...")
- Summaries ("So in conclusion...")
- Filler words ("essentially", "basically", "in order to")
- Padding ("Let me know if you need anything else!")

## Token Savings

| Style | Example | Est. Tokens |
|-------|---------|-------------|
| Default | "That's a great question! I'd be happy to help you understand how this works..." | ~20 |
| Caveman | "Here:" | ~2 |

**Result: ~50–70% fewer output tokens.**

## Use Cases

- Power users on paid API plans (cost reduction)
- High-volume automation pipelines
- Developers building token-efficient agents
- Anyone who values signal over noise

## How to Use

Paste the prompt at the start of any conversation, or add it to your system prompt.

Works with: Claude, ChatGPT, Gemini, and most instruction-following LLMs.
